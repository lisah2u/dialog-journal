## Usage

### Installation

Install latest from the GitHub
[repository](https://github.com/lisa/dialog-journal):

``` sh
$ pip install git+https://github.com/lisa/dialog-journal.git
```

or from [conda](https://anaconda.org/lisa/dialog-journal)

``` sh
$ conda install -c lisa dialog_journal
```

or from [pypi](https://pypi.org/project/dialog-journal/)

``` sh
$ pip install dialog_journal
```
